# 中国桥梁数据可视化系统

## 项目简介

这是一个基于 Vue.js + ECharts 的中国桥梁数据可视化项目，展示中国各类典型桥梁的工程信息、地理分布和技术特点。

## 快速启动

### 方式一：一键启动（推荐）

1. 双击运行 `启动桥梁系统.bat`
2. 自动打开浏览器访问 http://localhost:5173

### 方式二：开发模式

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 项目结构

```
my-project/
├── dist/                    # 打包后的静态文件
├── src/                     # 源代码
│   ├── assets/             # 静态资源（图片）
│   ├── components/        # 组件
│   ├── views/             # 页面视图
│   ├── store/             # 状态管理
│   └── router/            # 路由配置
├── public/                # 公共资源
├── 启动桥梁系统.bat        # 一键启动脚本
└── package.json
```

## 技术栈

- **框架**: Vue 3 + Composition API
- **构建工具**: Vite 8.x
- **可视化**: ECharts 6.x + vue-echarts
- **状态管理**: Pinia
- **路由**: Vue Router 5.x
- **UI组件**: Element Plus
- **样式**: Less

## 功能模块

| 模块 | 说明 |
|------|------|
| 首页 | 轮播展示 + 功能入口 |
| 数据大屏 | 桥梁类型可视化展示 |
| 地域分布 | 全国地图 + 省份钻取 |
| 结构分析 | 桥梁结构类型图表 |
| 技术特点 | 施工技术可视化 |
| 数据集管理 | 数据表格管理 |

## 重新打包

如果修改了代码，需要重新打包：

```bash
npm run build
```

打包后的文件会生成在 `dist` 目录中。

## 注意事项

- 启动脚本会自动打开浏览器
- 默认端口为 5173
- 确保已安装 Node.js 环境（用于 npm 命令）
- 如果 `npx serve` 启动失败，脚本会自动尝试使用 Python 启动
